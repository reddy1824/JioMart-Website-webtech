JioMartCart is a front-end e-commerce web application inspired by the JioMart platform.
It allows users to browse products, add items to a shopping cart, and simulate a checkout process.
The project is developed using HTML, CSS, and JavaScript to demonstrate client-side web development concepts.
It focuses on user interface design, cart management, and basic payment interaction.
This project is part of 5th Semester academic coursework and can be extended into a full-stack application.
No backend or database is used in the current version.
The project helps in understanding real-world e-commerce workflows.
